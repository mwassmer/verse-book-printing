# Interior PDF

The print-ready interior PDF is generated by the build system.

To generate it, run from the repository root:

**Windows (PowerShell):**
```powershell
.\build.ps1 -PrintReady
```

**Unix/macOS:**
```bash
./build.sh --print-ready
```

This produces `output/BookOfVerse-print.pdf` with:
- 7.25" x 10.25" paper (7" x 10" trim + 0.125" bleed)
- Crop marks on all pages
- Print-safe colors (no colored links)

Copy the generated PDF here as `interior.pdf` before submitting to the printer.
