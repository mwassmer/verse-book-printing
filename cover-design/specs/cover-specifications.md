### Spine Width Calculation

```
Spine Width = (Page Count × Paper Thickness) + (Board Thickness × 2)

For 469 pages on 60 lb paper:
- Paper: 469 × 0.0025" = 1.1725"
- Boards: 2 × 0.08" = 0.16"
- Total: ~1.33"

Confirm with your binder before finalizing!
```

## Embossing & Foil Specifications

### Foil Stamping Requirements

| Requirement | Specification |
|-------------|---------------|
| **Minimum line width** | 1mm (2.83 points) |
| **Minimum text size** | 8pt (12pt recommended) |
| **Font style** | Sans-serif, bold preferred |
| **Artwork format** | Vector (AI, EPS, PDF) or 600 DPI bitmap |
| **Color mode** | Solid black on white (for die) |
| **No gradients** | Foil is solid material only |
| **Max area per stamp** | 4" × 4" (check with binder) |

### Foil Color Options

| Finish | Code | Recommended Use |
|--------|------|-----------------|
| **Gloss Gold** | Standard | Title, primary elements |
| **Matte Gold** | Premium | Subtle elegance |
| **Gloss Silver** | Standard | Subtitle, secondary |
| **Matte Silver** | Premium | Modern feel |
| **Black Foil** | Specialty | High contrast on light leather |
| **Holographic** | Specialty | Not recommended for classic look |

### Embossing Specifications

| Type | Depth | Best For |
|------|-------|----------|
| **Blind deboss** | 0.5-1mm | Subtle patterns, textures |
| **Single-level emboss** | 1-2mm | Logos, simple shapes |
| **Multi-level sculptured** | 2-4mm | Complex 3D effects |
| **Combination** | Varies | Emboss + foil highlight |

## Production Notes

### Raised Spine Bands
- Traditional leather binding feature
- Typically 4-5 bands evenly spaced
- Text and logo positioned between/around bands
- Adds $50-150 to binding cost

### Gilded Edges
- Gold leaf applied to page edges (head, tail, fore-edge)
- Approximately $30-75 additional per book
- Creates premium, finished appearance
- Alternative: colored/painted edges

### Endpapers
- Heavy decorative paper inside covers
- Options: marbled, patterned, solid color
- Can be custom printed with design
- Typically 80-100 lb text weight
