## TIPS FOR BEST RESULTS

### For Embossing-Ready Logos:
1. Ask for "solid shapes only, no gradients"
2. Specify "minimum 1mm line thickness"
3. Request "high contrast, suitable for die cutting"
4. Use "vector style" in prompts

### For Realistic Mockups:
1. Specify "photorealistic" or "photography style"
2. Include lighting direction ("studio lighting", "soft shadows")
3. Describe viewing angle ("45-degree", "front view", "three-quarter")
4. Mention materials explicitly ("leather texture", "gold foil sheen")

### For Midjourney:
- Use `--ar 1:1` for logos, `--ar 4:3` or `--ar 3:4` for cover mockups
- Add `--style raw` for cleaner, less stylized results
- Use `--v 6` for latest version

### For DALL-E 3:
- Be very specific about what you DON'T want
- Describe the use case (embossing, foil stamping)
- Reference existing styles ("like classic O'Reilly books", "Addison-Wesley aesthetic")
